# Student Placement Prediction System

A production-style Machine Learning project to predict student placement status and expected salary package.

## Targets
- `placement_status` — Classification (Placed / Not Placed)
- `salary_package_lpa` — Regression (Expected salary in LPA)

## Tech Stack
- **ML**: Python, scikit-learn, pandas, numpy, matplotlib, seaborn
- **Backend**: FastAPI
- **Frontend**: React + Chart.js
